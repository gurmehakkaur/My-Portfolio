Project 1: Delivery Company Simulator

I developed an intelligent routing system for a local delivery company operating within a city. This system optimizes routes for three delivery trucks, ensuring efficient package deliveries while considering truck capacity limits. It calculates the shortest paths avoiding obstacles, and assigns packages to the truck that requires the least deviation from its route. The system also balances loads by comparing available weight and volume capacity, ensuring optimal resource utilization. If no truck can accommodate a package, it's stored for next-day delivery.

Tech Stack: C/C++
